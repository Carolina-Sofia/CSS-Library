Introduction

CSS (Cascading Style Sheets) controls the appearance of HTML elements.

It handles layout, colors, spacing, typography, and more.

This is just my library where I add things as I learn them.

Things that position
  • margin - Adds space outside an element.
  • padding (top, right, bottom, left) - Adds space inside an element, between the content and the border.
  • position - Defines how an element is placed in the document.
    • static: default, follows the normal flow.
    • relative: positioned relative to its normal position.
    • absolute: positioned relative to the nearest positioned ancestor.
    • fixed: positioned relative to the viewport (remains in place during scroll).
Things that give colors
  • background-color - Sets the background color of an element. Can be defined using color names, HEX codes, RGB, or HSL.
  • color - Sets the color of the text inside an element.
  • opacity - Adjusts the transparency of an element. Values range from 0 (fully transparent) to 1 (fully opaque).
Things that give borders
  • border - Adds a border around an element. Consists of width, style (solid, dashed, dotted, etc.), and color.
  • border-radius - Rounds the corners of an element’s border. Values can be set in pixels or percentages.
Things that link
  • <link> - only used on the head portion of html.
  • <a></a> anchor element that actually links to other pages.
  • Referer HTTP header contains information about the address or URL of a page that a user might be visiting from. This information can be used in analytics to track how many users from your page visit the link, for example. Setting the rel attribute to noreferrer omits this information from the HTTP request.
  • target - defines where the link is opened.
    • _blank: opens the document in a new window or tab.
    • _self: default, opens the document in the same window/tab as it was clicked.
    • _parent: opens the document in the parent frame.
    • _top: opens the document in the full body of the window.
Grid

You start by "activating" this feature with display: grid;. Offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning. Key properties:

  • row-gap: sets the space between the rows.
  • grid-column: shorthand for grid-column-start and grid-column-end. The grid-column property tells the grid item which grid line to start and end at.
  • flex-direction: can be row, column, row-reverse, or column-reverse.
Images
  • loading="" - can be set to lazy to tell the browser not to fetch the image resource until it is needed (as in, when the user scrolls the image into view). As an additional benefit, lazy loaded elements will not load until the non-lazy elements are loaded - this means users with slow internet connections can view the content of your page without having to wait for the images to load.
  • alt="" - images should always have an alternative text to help screen-reading tools describe images to visually impaired readers and allows search engines to better crawl and rank the website.
Typography
  • font-family - Defines the font for the text inside an element. It's recommended to include a fallback font.
  • font-size - Sets the size of the text inside an element. Values can be in pixels, em, rem, or percentage.
  • font-weight - Specifies the thickness or boldness of the text (e.g., normal, bold, or numerical values).

Flexbox

You start by "activating" this feature with display: flex;. Provides a flexible layout for aligning items in rows or columns. Key properties:

  • justify-content: controls horizontal alignment.
  • align-items: controls vertical alignment.
  • flex-direction: can be row, column, row-reverse, or column-reverse.
Overflow
  • visible - Default. Content that overflows is visible outside the element’s box.
  • hidden - Cuts off content that overflows the element's box, hiding it from view.
  • scroll - Adds scrollbars when content overflows, allowing the user to scroll.
Variables

The main advantage of using variables is the ability to quickly change many values in your stylesheet by just changing the value of a variable. The structure of the variable looks like this: var(--variable-name, value); Afterwards, we can just use var(--variable-name) as the value for the properties we want to control with the variable.
