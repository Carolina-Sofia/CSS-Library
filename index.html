Introduction

CSS (Cascading Style Sheets) controls the appearance of HTML elements.

It handles layout, colors, spacing, typography, and more.

This is just my library where I add things as I learn them.

Things that position
  • Margin - Adds space outside an element.
  • Padding (top, right, bottom, left) - Adds space inside an element, between the content and the border.
  • Position - Defines how an element is placed in the document.
    • static: default, follows the normal flow.
    • relative: positioned relative to its normal position.
    • absolute: positioned relative to the nearest positioned ancestor.
    • fixed: positioned relative to the viewport (remains in place during scroll).
  • Flexbox - Provides a flexible layout for aligning items in rows or columns. Key properties:
    • justify-content: controls horizontal alignment.
    • align-items: controls vertical alignment.
Things that give colors
  • Background-Color - Sets the background color of an element. Can be defined using color names, HEX codes, RGB, or HSL.
  • Text Color - Sets the color of the text inside an element.
  • Opacity - Adjusts the transparency of an element. Values range from 0 (fully transparent) to 1 (fully opaque).
Things that give borders
  • Border - Adds a border around an element. Consists of width, style (solid, dashed, dotted, etc.), and color.
  • Border-Radius - Rounds the corners of an element’s border. Values can be set in pixels or percentages.
Typography
  • Font-Family - Defines the font for the text inside an element. It's recommended to include a fallback font.
  • Font-Size - Sets the size of the text inside an element. Values can be in pixels, em, rem, or percentage.
  • Font-Weight - Specifies the thickness or boldness of the text (e.g., normal, bold, or numerical values).
Box Model
  • Content - The actual content of the element (e.g., text, image).
  • Padding - Space between the content and the border.
  • Border - The edge around the padding. Can have width, style, and color.
  • Margin - Space outside the border, separating the element from other elements.
Overflow
  • Visible - Default. Content that overflows is visible outside the element’s box.
  • Hidden - Cuts off content that overflows the element's box, hiding it from view.
  • Scroll - Adds scrollbars when content overflows, allowing the user to scroll.
